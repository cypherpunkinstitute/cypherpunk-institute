# <i class="fa-brands fa-linux fa-lg"></i> Sistemas Operativos para PC

Cuando se trata de proteger nuestra privacidad y soberanía digital, el primer paso esencial que debemos tomar es cambiar nuestro sistema operativo. Este es el software principal que ejecuta todas las aplicaciones y que se comunica directamente con el hardware de nuestro equipo. En sistemas operativos propietarios como Windows o macOS, no tenemos visibilidad sobre cómo funciona internamente el sistema debido a que su código es cerrado. Esto significa que debemos confiar en que los desarrolladores no introduzcan puertas traseras o mecanismos de monitorización de actividades para las que no hemos dado nuestro consentimento. Eso sin contar la basta cantidad de datos que recopilan de nosotros a través de telemetría, datos de diagnóstico o servicios en la nube, para las que hemos tenido que dar nuestro consentimiento a cambio de poder usar su software.

Por estas razones, la mejor alternativa es utilizar un sistema operativo libre y de código abierto como Linux, que respeta la privacidad del usaurio y le devulve el control de su sistema. Con Linux no solo podemos ver el código y entender completamente su funcionamiento, sino que también tenemos la capacidad de eliminar cualquier funcionalidad indeseada. Existen distribuciones de Linux tanto para propósito general como para enfoques específicos en privacidad y seguridad, aunque estas últimas son más adecuadas para situaciones concretas y no se suelen instalar como sistema operativo principal. Usar Linux nos brinda un control total sobre nuestro sistema y nuestros datos, proporcionandonos una verdadera libertad y soberanía digital.
